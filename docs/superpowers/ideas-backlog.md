## A급 — 반짝이

### 메뉴바 글리프
macOS 메뉴바 아이콘에 다음 일정 + P0 개수. 클릭으로 창 복원.

- **Depends on:** Tauri `tray-icon` + 윈도우 show/hide

### 프로젝트 "마지막 터치" 뱃지
카드 모서리에 `3일 전` · `2주 전`. 방치된 프로젝트가 눈에 박히게.

- **Depends on:** `projects` 테이블에 `last_touched_at` (현재 있는지 확인 필요)

### 메모 완료 체크
메모 우상단 체크박스. 체크 시 흐려지고 하단 섹션으로. 삭제와 분리.

- **Why:** 지금은 완료/진행중 구분이 없어 색으로 편법 분류
- **Depends on:** `memos.completed_at` 컬럼 추가 + UI

### ⌘K 스마트 감지
클립보드에 URL/날짜/숫자 있으면 팔레트 상단에 "이 링크로 메모 만들기" 1줄 추천.

- **Depends on:** 클립보드 읽기 권한 + 팔레트 훅
